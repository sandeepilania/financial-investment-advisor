Title: Real Returns vs Nominal Returns
Category: market_basics
Tags: real_returns, inflation_adjusted_returns, investment_performance
Risk Profile: all
Target Audience: beginner
Investor Horizon: long
Last Updated: 2026-03

Key Points:
- Nominal returns represent raw investment gains.
- Real returns account for inflation.
- Inflation can significantly reduce real wealth.
- Investors should evaluate performance using real returns.

Content:
Nominal returns refer to the total return of an investment
before accounting for inflation. While nominal returns may
appear attractive, they do not always reflect true purchasing
power gains.

Real returns adjust nominal returns for inflation. This
measure provides a more accurate view of how an investment
affects an investor's wealth.

For example, if an investment generates a 7% return while
inflation is 3%, the real return is approximately 4%.

Understanding the difference between real and nominal
returns helps investors evaluate whether their portfolios
are truly growing in value.

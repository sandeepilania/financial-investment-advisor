Content:
Modern Portfolio Theory (MPT) is an investment framework
introduced by economist Harry Markowitz. The theory
suggests that investors can build portfolios that
maximize expected returns for a given level of risk.

A key concept in MPT is diversification. By combining
assets that do not move in perfect correlation with
each other, investors can reduce portfolio volatility
without sacrificing returns.

The goal of portfolio construction under MPT is to
create a combination of assets that lies on the
efficient frontier.
